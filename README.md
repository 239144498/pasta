# pasta
Pasta is clipboard application for ios devices.

## Açıklama

YMGK2 Uygulamasının telefon ve bilgisayarlarda çalışmasını sağlayan istemci uygulaması.

## Starting

### Requirements

* Xcode 13

### Runing

* Not requires any specific steps, clone and run.
